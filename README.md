# Add-to-Cart Demo App

A simple front-end application showcasing an add-to-cart functionality, built using modern web technologies.

## Project Description

This project implements a basic “add to cart” workflow for a product list, allowing users to select items and view their cart state.  
Designed to demonstrate the fundamentals of front-end state management, UI updates, and user interaction.

## Live Demo

https://pr9-addtocart.vercel.app/

## Features

- Display a list of products.  
- Add products to the cart.  
- View cart contents and quantity.  
- Remove or reduce item quantity from cart (if implemented).  
- Responsive UI (assuming).  
- Demo‐friendly and ready for customization.

## Tech Stack

- HTML5, CSS3
- BootStrap5
- JavaScript   
- Deployment via Vercel
